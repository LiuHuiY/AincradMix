### 简单的介绍    
>- 全称    
>>R_Sp[osu!]的全称是Remixes Sword art online Project for osu!game    
>- 设计理念    
>>基于在整个《刀剑神域》系列番剧中出现的所有UI的基础上，将更为现代的设计理念与osu!游戏风格相混合    
>- R_Sp[osu!]的好处都有啥？    
>>可定制化程度极高    
>>良好的打图手感    
>>简洁而不失美观的视觉体验    
>>延续了刀剑神域系列番剧中UI的风格    
>>支持多种`类osu!`项目    
>>...    
>- 较为完整的支持    
>>McOsu,FPoSu,Opsu    
>- 目前不支持    
>>osu!droid,iosu!    
>- 不支持    
>>osu!lazer
